# 🏥 Hospital Management Analytics Dashboard

An end-to-end SQL and Excel analytics project that analyzes hospital operations through business KPIs and interactive dashboards.

## 📌 Overview

This project simulates a hospital management system by transforming raw relational data into actionable business insights using **MySQL** and **Microsoft Excel**.

## 🚀 Tech Stack

- MySQL
- MySQL Workbench
- Microsoft Excel

## ✨ Key Features

- Designed a relational database with **18 interconnected tables**
- Performed data cleaning and standardization using SQL
- Built **38 business SQL queries** for hospital analytics
- Created **2 interactive Excel dashboards**
- Generated executive KPIs and patient insights

## 📊 Executive Dashboard

Provides a high-level view of hospital performance through:

- Total Patients & Admissions
- Total Revenue
- Average Bill & Stay Length
- Bed Occupancy Rate
- Monthly Revenue & Admission Trends
- Department-wise Revenue Analysis
